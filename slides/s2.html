Connecting to a server
  • Verify SSH client is installed: 
    ssh -V
  • Connect to your server: 
    ssh <user>@<your_ip>
  • Try to run 
    whoami

Remote command execution and PTY

  • Plain SSH connection - PTY allocated
    ssh <user>@<your_ip>
  • SSH command execution - No PTY 
    ssh <user>@<your_ip> <command>
  • Force enabled PTY
    ssh -t <user>@<your_ip> <command>
  • Force disabled PTY
    ssh -T <user>@<your_ip>
PTY = Pseudo Teletype

Download/Upload files: SCP

  • Download files 
    scp [-r] hostname:remote_path local_path
  • Upload files 
    scp [-r] local_path hostname:remote_path
