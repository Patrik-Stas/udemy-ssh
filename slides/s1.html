Running server: The options

  • On your localhost
    • Free
    • More effort to set up
    • Need to download large file
    • Might slow dow computer
    • Further away from real use-case
  • In cloud - recommended
    • Simple to set up
    • Typical use-case
    • Costs 5$

    Creating server with DigitalOcean
    • Open https://www.digitalocean.com
    • Create new account
    • Sign in
    • Create new virtual server
    • Verify SSH connection

    Creating server locally with Virtualbox
    • Download virtualbox: https://www.virtualbox.org
    • Install virtualbox
    • Download minimal CentOS image: https://www.centos.org/download/
    • Create new Virtual Machine from the image
    • Install CentOS on Virtual Machine
    • Verify SSH connection
